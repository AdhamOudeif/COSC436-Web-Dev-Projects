Weather Tracker

Stack: Front End Using Vue.js, Javascript for backend | Role: Lead Developer | Technologies: WeatherStack API, Google API


Description:

Using Vue.js Framework to create a task tracking app and pull data and info using Google and WeatherStack APIs.
Google API is used to track the location using coordinates, and then sends that info to the WeatherStack app
to gauge how the weather will be on that day.
