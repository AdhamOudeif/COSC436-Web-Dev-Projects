Mocha Managment

Stack: LAMP | Role: Developer | Technologies: AWS, Vs Code, Xampp, AI


Description:

Built an in-house web app using XAMPP that streamlines activities such as scheduling and inventory management, which are imperative to keep a business organized and running.
The app allows for employers to manage schedules and inventory either using AI or manually.
Employees are able to view their schedules, request days off, and update the inventory table.
Worked with a team across the software development life cycle including developing ER diagrams, interface
diagrams, developing prototypes, building the database, set up testing framework (codeception), maintenance and
refactoring, and deployment.
